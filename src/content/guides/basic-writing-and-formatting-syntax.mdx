# Headings

To create a heading, add one to two <kbd>#</kbd> symbols before your heading text. The number of <kbd>#</kbd> you use will determine the hierarchy level and typeface size of the heading.

<Figure withCodeblock>
  <div>
    <h1>Heading 1</h1>
    <div/>
    <h2>Heading 2</h2>
  </div>
</Figure>

```text copy
# Heading 1
# Heading 2
```

# Styling Text

You can indicate emphasis with bold, italic, strikethrough, subscript, or superscript text in comment fields and .md files.

<Figure withCodeblock>
  <div>**Bold**</div>
  <div>*Italic*</div>
  <div>~~Strikethrough~~</div>
  <div><sub>Subscript</sub></div>
  <div><sup>Superscript</sup></div>
  <div><kbd>⌘</kbd><kbd>K</kbd></div>
</Figure>

```text
**Bold**
*Italic*
~~Strikethrough~~
<sub>Subscript</sub>
<sup>Superscript</sup>
<kbd>⌘</kbd><kbd>K</kbd>
```

# Quoting Text

You can quote text with a <kbd>></kbd>.

<Figure withCodeblock>
  Text that is not a quote
  > Text that is a quote
</Figure>
```text
Text that is not a quote
> Text that is a quote
```

# Links

You can create an inline link by wrapping link text in brackets, and then wrapping the URL in parentheses.

<Figure withCodeblock>
  [Link](https://www.youtube.com/watch?v=dQw4w9WgXcQ)
</Figure>
```text
[Link](https://www.youtube.com/watch?v=dQw4w9WgXcQ)
```

# Lists

You can make an unordered list by preceding one or more lines of text with <kbd>-</kbd>.

<Figure withCodeblock>
  - George Washington
  - John Adams
  - Thomas Jefferson
</Figure>
```text
- George Washington
- John Adams
- Thomas Jefferson
```
To order your list, precede each line with a number.

<Figure withCodeblock>
  1. James Madison
  2. James Monroe
  3. John Quincy Adams
</Figure>
```text
1. James Madison
2. James Monroe
3. John Quincy Adams
```

# Footnotes

You can add footnotes to your content by using this bracket syntax <kbd>`[^1]`</kbd> and then defining the footnote at the bottom of your content.


<Figure withCodeblock>
  <div>The invention of the printing press revolutionized the spread of information in Europe during the 15th century.[^2]</div>

  [^2]: Johannes Gutenberg is credited with inventing the movable-type printing press around 1440 in Mainz, Germany.
</Figure>
```text
The invention of the printing press revolutionized the spread of information in Europe during the 15th century.[^2].

[^2]: Johannes Gutenberg is credited with inventing the movable-type printing press around 1440 in Mainz, Germany.
```

# Inline Code

Append <kbd>`{:lang}{:js}`</kbd> (e.g. <kbd>`{:js}{:js}`</kbd>) to the end of inline code to highlight it like a regular code block.

<Figure withCodeblock>
  To useEffect or not to `useEffect(){:tsx}`, that is the question.
</Figure>
```text
This is an array `[1, 2, 3]{:js}` of numbers 1 through 3.
```


# Code Blocks

To format code or text into its own distinct block, use triple backticks.
 
<Figure withCodeblock>
```tsx
function feature() {
  return "It works in my environment.";
}
```
</Figure>
```text
```tsx  
function feature() {
  return "It works in my environment.";
}
```‎
```

# Media

You can embed images, videos, and other media in your markdown content. You can also add captions to your images via the <kbd>`<Image/>`</kbd> component.

<Figure withCodeblock>
<div className="flex flex-row gap-4">
  <Image src='https://avatar.vercel.sh/sylph' alt='Placeholder' caption='Avatar'/>
</div>
</Figure>
```text
<Image src='https://avatar.vercel.sh/sylph' alt='Placeholder' caption='Avatar'/>
```

# Tables

You can create tables by assembling a list of words and dividing them with hyphens <kbd>-</kbd> (for the first row), and then separating each column with a pipe <kbd>|</kbd>.

<Figure withCodeblock>
| First Header  | Second Header |
| ------------- | ------------- |
| Content Cell  | Content Cell  |
| Content Cell  | Content Cell  |
</Figure>
```text
| First Header  | Second Header |
| ------------- | ------------- |
| Content Cell  | Content Cell  |
| Content Cell  | Content Cell  |
```
